**git小白个人操作**

# 首先得确保电脑装有GIT
> debian系安装命令：sudo apt install git  
> arch系安装命令：sudo pacman -S git  


# 运行下面两个命令（git全局设置）
>git config --global user.name "用户名"   
>git config --global user.email "用户邮箱"

# vscode操作
+ 确保操作文件为git仓库(手动创建或者git clone拉取的均为git仓库)
+ vscode打开仓库文件夹
+ 对文件进行操作
